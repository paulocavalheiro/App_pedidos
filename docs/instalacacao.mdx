## Requisitos

Antes de começar, você vai precisar do Node.js e do Docker.

## Passo a passo

<Steps>
  <Step title="Clone o repositório">
    ```bash git clone https://github.com/seu-usuario/app-pedidos.git ```
  </Step>
  <Step title="Configure o Backend">
    Entre na pasta `backend` e rode `npm install`.
  </Step>
  <Step title="Inicie o Frontend">
    Entre na pasta `frontend` e rode `npm run dev`.
  </Step>
</Steps>

<Info>
  Dica: Certifique-se de que o banco de dados está rodando antes de iniciar o
  backend.
</Info>
